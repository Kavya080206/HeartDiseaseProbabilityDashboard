# Heart Disease Probability Dashboard

This project predicts the probability of heart disease using Machine Learning and Streamlit.

## Features
- Real dataset based prediction
- Interactive Streamlit dashboard
- Data visualization
- Risk probability output
- Deployment ready

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib

## Run the app
```bash
python -m streamlit run app.py
